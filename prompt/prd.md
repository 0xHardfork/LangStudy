# LangStudy 需求文档 (PRD)

## 1. 英语语法与文章分析 (追加功能)

### 1.1 功能概述
该功能允许用户输入或粘贴任意英文段落/文章，系统自动按句子进行边界识别与拆分。用户可以通过点击文章中的任意句子，呼出详细的语法结构解析面板，辅助深度英语阅读与语法学习。

### 1.2 核心业务流程
1. **文章输入与保存**：
   - 用户在“文章语法分析”界面中，通过文本框输入/粘贴英文文章。
   - 点击“开始分析”，后端将文章保存至数据库（用于历史复习），并将文章拆分为多个句子展示在前端。
2. **段落级展示**：
   - 前端将整篇文章渲染出来，但将每个句子识别为一个可点击的交互块。
3. **句级交互式语法分析**：
   - 用户点击文章中的某个句子，下方的“语法解析面板”显示 AI 实时生成的结构分析：
     - **成分高亮着色**：将该句子拆分为多个连续的不重叠片段（Segments），每个片段根据语法成分（如：主语、谓语、宾语、定语从句、状语等）使用不同颜色高亮。
     - **成分悬停提示**：鼠标悬停在特定高亮片段上时，展示该片段的具体语法作用说明。
     - **句型模式徽章**：标明句子的核心句型（例如：`SVO (主谓宾) + Attributive Clause (定语从句)`）。
     - **释义与发音**：显示准确的中文翻译，并提供发音播放功能（可使用 Web Speech API 或后端合成语音）。
     - **核心语法点解析**：以列表形式展示本句中使用的 2-3 个核心语法知识点及其详细解析。
     - **核心词汇提取**（可选）：从句子中自动提取出 3-5 个重点词汇，展示其音标、词性及在本句上下文中的释义。

### 1.3 交互设计规范
- **整句结构拆分着色规范**：
  - 主语 (Subject): 蓝色 (`#60a5fa`)
  - 谓语 (Verb/Predicate): 紫色 (`#a78bfa`)
  - 宾语/表语 (Object/Predicative): 绿色 (`#34d399`)
  - 定语/定语从句 (Attribute/Clause): 品红/粉色 (`#f472b6`)
  - 状语/状语从句 (Adverbial/Clause): 黄色/橙色 (`#fbbf24`)
  - 其它修饰/介词短语 (Others): 灰色/次级色 (`#94a3b8`)
- **移动端与响应式适配**：
  - 左右布局（左边长文，右边解析面板）在宽屏上显示；在窄屏/移动端自动折叠为上下布局。

### 1.4 后端接口设计 (Go)
1. `POST /api/v1/grammar/analyze`：
   - 输入：`{ "text": "文章内容..." }`
   - 行为：服务端按句号/问号/感叹号进行文本句子切分，调用大模型（JSON 模式）对各句进行成分、翻译、语法点及生词分析，保存并返回数据。
2. `GET /api/v1/grammar/history`：
   - 返回当前用户分析过的历史文章列表。
3. `GET /api/v1/grammar/article/:id`：
   - 获取某篇历史文章及其句子的完整预分析缓存，避免重复调用 AI。
